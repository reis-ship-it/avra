## üéØ **Decision-Making Process**

### **Step 1: Always Check OUR_GUTS.md First**
Before making ANY decision about SPOTS, ask:

- Does this support "Belonging Comes First"?
- Does this maintain "Privacy and Control Are Non-Negotiable"?
- Does this uphold "Authenticity Over Algorithms"?

### **Step 2: Core Values Checklist**
Every decision must pass these checks:

#### **‚úÖ Belonging Comes First**
- Does this help people find places where they truly feel at home?
- Does this create authentic connections and community?
- Does this support "know you belong"?

#### **‚úÖ No Agenda. No Politics. No Pay-to-Play**
- Is this neutral and open?
- Are suggestions never for sale?
- Does this help users find what fits THEM, not someone else's agenda?

#### **‚úÖ Authenticity Over Algorithms**
- Is this powered by real user data and preferences?
- Does this help discover what users will actually enjoy?
- Is this not driven by advertising dollars or hidden motives?

#### **‚úÖ Privacy and Control Are Non-Negotiable**
- Does this respect user data ownership?
- Is user consent always required?
- Is data encrypted and secure?
- Does the user maintain control?

#### **‚úÖ Effortless, Seamless Discovery**
- Is this seamless and up-to-date?
- Is it not overbearing or intrusive?
- Does it let users enjoy the moment while we handle the rest?

#### **‚úÖ Personalized, Not Prescriptive**
- Is this suggestive, not predictive?
- Is the user always in control?
- Can users adjust their preferences?

#### **‚úÖ Community, Not Just Places**
- Does this bring people together?
- Does this help connect with people, events, and experiences?
- Is this for everyone, everywhere, and every community?
