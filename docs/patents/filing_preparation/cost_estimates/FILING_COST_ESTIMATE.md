## üí° Cost-Saving Strategies

1. **Small Entity Status:** If eligible, saves 50% on USPTO fees
2. **Micro Entity Status:** If eligible, saves 75% on USPTO fees
3. **Batch Filing:** Some attorneys offer discounts for multiple patents
4. **Provisional Patents:** File provisional first ($80-$320), then non-provisional within 12 months
5. **In-House Preparation:** Reduce attorney fees by preparing documents in-house (with legal review)
