# SPOTS Quantum Package

Quantum services, quantum matching, and quantum vibe engine for SPOTS.

## Overview

This package contains all quantum-related services and functionality for SPOTS, including:

- **Quantum Entanglement Services** - Multi-entity quantum entanglement matching
- **Quantum State Services** - Location and timing quantum state conversion
- **Quantum Learning Services** - Outcome-based learning and ideal state learning
- **Real-Time User Calling** - Dynamic user calling based on evolving entangled states
- **Meaningful Connection Metrics** - Quantum-based connection quality metrics

## Usage

```dart
import 'package:spots_quantum/spots_quantum.dart';
```

## Dependencies

- `spots_core` - Core models and utilities
- `spots_ml` - ML models and training pipelines
