# SPOTS Knot Package

Knot theory services, knot weaving, and knot fabric for SPOTS.

## Overview

This package contains all knot theory-related services and functionality for SPOTS, including:

- **Personality Knot Service** - Generate personality knots from personality profiles
- **Knot Weaving Service** - Create braided knots from two personality knots
- **Knot Fabric Service** - Create and analyze knot fabrics for community representation
- **Entity Knot Service** - Generate knots for all entity types (people, events, places, companies)
- **Cross-Entity Compatibility** - Calculate compatibility using knot theory
- **Integrated Knot Recommendation Engine** - Combines quantum and topological compatibility

## Usage

```dart
import 'package:spots_knot/spots_knot.dart';
```

## Dependencies

- `spots_core` - Core models and utilities
- `spots_quantum` - Quantum services (for integrated recommendations)
